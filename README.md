
Consolist
=========

![](https://dl.dropboxusercontent.com/u/37804131/github/consolist.png)

Outputs a bitmap image to a browser's developer console or a terminal.

Tested on:

- Web browsers
    - Google Chrome
    - Safari
    - Firefox
- Terminal emulators
    - iTerm2

Usage
-----

Please see `examples` and `consolist.d.ts` for usage.
